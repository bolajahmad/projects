- Developed a full-stack management dashboard enabling users to manage subscriptions while providing merchants with payment analytics and billing administration.
- Designed the protocol architecture to minimize cross-chain settlement latency while preserving verifiable payment execution and user ownership of funds.

### [Decentralized Agentic Payments](https://github.com/bolajahmad/decentralized-agentic-payment) - Machine-native payment infrastructure for autonomous agents

Designed a payment protocol enabling AI agents to securely purchase APIs
and digital services through user-controlled wallet authorization and
verifiable on-chain settlement.

- Implemented Solidity smart contracts supporting session-scoped authorization and programmable spending limits for autonomous agents.
- Designed a proof-based settlement mechanism reducing on-chain transaction overhead through batched payment commitments while preserving payment verifiability.
- Built JavaScript SDKs and CLI tooling allowing wallet owners to configure, monitor and audit agent payment activity in real time.
- Integrated the protocol with LLM-driven agent workflows, enabling autonomous payment execution through conversational interfaces and developer tooling.

### [PVM Runtime Precompiles](https://github.com/bolajahmad/dot-crypto-precompiles-primities) - Developer tooling for Substrate-native runtime cryptography.

Built developer tooling exposing native runtime cryptographic precompiles to
Solidity developers, enabling access to advanced cryptographic operations
without reimplementing them inside smart contracts.

- Developed Rust libraries implementing encoding, decoding and payload generation for BLS12-381, Schnorr signatures and additional runtime cryptographic primitives.
- Built Solidity helper libraries exposing strongly typed interfaces for runtime precompiles, significantly simplifying contract integration with native runtime functionality.
- Designed reusable CLI tooling generating deterministic payloads, test vectors and validation data for smart contract development and protocol testing.
- Improved the developer experience for Revive-compatible runtimes by providing a complete tooling layer spanning Solidity libraries, Rust crates and command-line utilities.

<small>
  *Expertise: Runtime Engineering, Cryptography, Rust, Solidity, Developer
  Tooling, Polkadot SDK*
</small>
