# Overview

This project explored extending the developer experience of
Polkadot AssetHub by allowing Solidity contracts to initiate
cross-chain operations through precompile interfaces.

The work focused on understanding how XCM execution paths
could be exposed to EVM developers without requiring
native runtime code or ink! smart contracts.

# Motivation

One of the largest barriers to adoption of advanced
Polkadot functionality is the requirement to understand
runtime-level development.

Precompiles provide an opportunity to expose runtime
capabilities through familiar EVM interfaces.
